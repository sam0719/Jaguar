## 對齊時遇到的情況

1. 位置**都**有移動,fitness相同,初始步伐減半跳出狩獵減速
2. 狩獵初期右左看後，左右位置其中有一邊與中間相等即可(對齊文件寫的是兩邊都要與中間相同)
3. 狩獵減速時左右位置和中間**都**不一樣，並且左右fitness與中間**都**一樣，直接結束狩獵
4. 跳脫距離是double類型,文檔裡實際上是float(用float的情況下 Rastrigin 30維30次實驗，第13次次數少了570次)
